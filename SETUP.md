# Quick Setup Guide

## Prerequisites
- Node.js (v16 or higher)
- npm or yarn

## Quick Start

1. **Install all dependencies:**
   ```bash
   npm run install-all
   ```

2. **Start the application:**
   ```bash
   npm run dev
   ```

3. **Access the application:**
   - Frontend: http://localhost:3000
   - Backend: http://localhost:5000

4. **Login with default credentials:**
   - Username: `admin`
   - Password: `admin123`
